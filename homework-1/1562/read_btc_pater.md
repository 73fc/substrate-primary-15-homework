# 读比特币白皮书感慨
```shell
    读了比特币的白皮书后，了解到电子货币系统至少需要四个模块组成：P2P 网络协议、分布式一致算法（共识机制）、加密签名算法、账户与存储模型。
    其中，工作量证明的这种共识机制给人大为震撼。

    比特币采用的是工作量证明（Proof of Work, PoW）机制，这是一种通过解决复杂数学问题来验证交易并添加新区块到区块链上的方法。
    以下是对比特币共识机制读后感的一些可能的思考，以及它对我们生活的意义。
    * 首先，它重塑了现在社会的金融信任体系。在传统的金融系统中，信任通常建立在中央机构或第三方的基础上。比特币的共识机制通过技术手段重塑了信任，使得交易可以在没有中央权威的情况下进行验证。 
这点对我们社会的贸易体系产生了重大的影响,重塑了我们社会的贸易体系。
    * 同时，它对参与到虚拟货币中的旷工进提高济激励。比特币的共识机制通过区块奖励和交易费为矿工提供了经济激励，这促进了网络的维护和发展，提高了在虚拟网络中的活跃度。
    * 最后，也是争议最大的一点。 很多人认为这种共识机制，会导致大量的资源消耗，导致不必要的电力能源浪费，以及对我们生存的环境造成影响。
但我认为可以从另一个角度看问题，我们国家东西部发展不平衡，西部有大量的清洁可用能源, 东部的企业发展又缺电, 对我们东部的企业发展形成了一定的压力。
我们完全可以利用现有的条件，采用西电东输方案，西部发电，东部计算，增加老百姓的收入。同时东部的企业也可以使用大量廉价的电力，产生更多的经济效益。

以上我对比特币共识机制的一些想法。

```
